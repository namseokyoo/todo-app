# Todo App by SidequestLab

간단하고 아름다운 Todo List 웹 애플리케이션입니다.

## 기능

- 할 일 추가/삭제/완료 체크
- LocalStorage로 데이터 영속성
- 반응형 디자인
- 완료된 항목 일괄 삭제

## 기술 스택

- **Frontend**: Next.js 14 + TypeScript
- **스타일링**: Tailwind CSS
- **상태관리**: React Hooks + LocalStorage
- **배포**: Vercel

## 시작하기

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build

# 프로덕션 서버 실행
npm run start
```

## 배포

Vercel에 자동 배포됩니다. `main` 브랜치에 푸시하면 자동으로 배포됩니다.
